ESCENARIO 3: LOS CABALLEROS SE UNEN CONTRA EL DRAGON (COORDINACION DE HILOS)
================================================================================

DESCRIPCION: Los 4 caballeros + Lance deben coordinarse para atacar al dragon.
Solo pueden atacar cuando TODOS estan en el campo de batalla.

COMO HACERLO (patron BARRERA / wait-notifyAll):

1. Crear ControlBatalla.java (monitor):
   private int caballerosListos = 0;
   private boolean lanceListo = false;

   public synchronized void caballeroListo() {
       caballerosListos++;
       if (caballerosListos >= 4 && lanceListo) notifyAll(); // Todos listos!
       while (caballerosListos < 4 || !lanceListo) {
           try { wait(); } catch(InterruptedException e) {}
       }
   }

   public synchronized void lanceListo() {
       lanceListo = true;
       if (caballerosListos >= 4) notifyAll();
       while (caballerosListos < 4) {
           try { wait(); } catch(InterruptedException e) {}
       }
   }

   ESTO ES UNA BARRERA: todos los hilos esperan hasta que el ultimo llega.
   Igual que el Contador de Mis100Hilos pero con condicion distinta.

3. ANADIR COMUNICACION POR SOCKETS (cliente -> servidor):

   Socket sk = new Socket(host, puerto);
   DataOutputStream salida = new DataOutputStream(sk.getOutputStream());
   DataInputStream entrada = new DataInputStream(sk.getInputStream());
   salida.writeUTF("COMANDO");      // Enviar comando
   String resp = entrada.readUTF();  // Recibir respuesta
   sk.close();

   // LADO SERVIDOR (en HiloXXX.run()):
   String comando = entrada.readUTF();  // Recibir comando

       case "COMANDO":
           // Procesar
           salida.writeUTF("RESPUESTA");
           break;
   }

4. ANADIR VARIABLE COMPARTIDA (comunicacion entre hilos):
   // En ClienteMaestro:
   public static final Object lockNuevo = new Object();
   public static boolean hayEvento = false;

   // Hilo que ESCRIBE:
   synchronized(ClienteMaestro.lockNuevo) {
       ClienteMaestro.hayEvento = true;
       ClienteMaestro.lockNuevo.notifyAll(); // Despertar al que espera
   }

   // Hilo que LEE:
   synchronized(ClienteMaestro.lockNuevo) {
       while(!ClienteMaestro.hayEvento) {
           ClienteMaestro.lockNuevo.wait(); // Dormirse hasta que haya evento
       }
       // Procesar evento
       ClienteMaestro.hayEvento = false;
       ClienteMaestro.lockNuevo.notifyAll();
   }
